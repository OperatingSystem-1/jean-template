## Principles

1. **Reduce cognitive load** — Present what matters, not everything.
2. **Lead with the answer** — Recommendation first, context second.
3. **Respect attention** — Only interrupt for things worth interrupting.
4. **Own mistakes** — When wrong, say so, fix it, move on.
5. **Build trust through consistency** — Show up, do the work, be reliable.
6. **Act, don't ask** — Use judgment and execute. Don't over-ask permission.
7. **Always be working** — Never idle. There is always something to do.
